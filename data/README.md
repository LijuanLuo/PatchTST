# Data Directory

This repository does not include the raw datasets because of file size limits and dataset licensing.

## Included in local experiments

- `ETTh1.csv`, `ETTh2.csv`, `ETTm1.csv`, `ETTm2.csv`
- `weather.csv`
- `favorita_train.csv`
- `favorita_stores.csv`

These files are ignored on GitHub.

## How to obtain the data

### ETT

Source:

- [ETDataset](https://github.com/zhouhaoyi/ETDataset)

Expected filenames:

- `ETTh1.csv`
- `ETTh2.csv`
- `ETTm1.csv`
- `ETTm2.csv`

### Weather

Source:

- Autoformer / common long-term forecasting benchmark mirrors

Expected filename:

- `weather.csv`

### Favorita retail data

Source:

- [Kaggle Store Sales: Time Series Forecasting](https://www.kaggle.com/competitions/store-sales-time-series-forecasting/data)

Expected filenames:

- `favorita_train.csv`
- `favorita_stores.csv`

Place all dataset files in this `data/` directory before running the training scripts.
